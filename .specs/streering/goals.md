## Goals

1. **Authentication System** - Login menggunakan dummyjson.com auth API, menyimpan token, dan protected routes
2. **Dashboard Layout** - Sidebar navigasi, navbar dengan user info, dan main content area
3. **Product Management (CRUD)** - List, search, detail, add, edit, delete produk menggunakan dummyjson.com products API
4. **Reusable Components** - Komponen yang bisa dipakai ulang (Button, Input, Card, Table, Modal, dll)
5. **State Management** - Menggunakan Zustand untuk auth state dan product state
6. **Responsive Design** - Tampilan responsive untuk desktop dan mobile
7. **Deploy & Public Repo** - Hosting app dan push ke public GitHub repo
